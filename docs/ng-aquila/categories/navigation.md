# Navigation Components

## Overview

Navigation components help users move through your application, access different sections, and understand their current location within the site structure.

**Common Use**: Navigation menus, action menus, context-sensitive options

**Quick Description**: Tabbed navigation interface

**File**: [breadcrumb.md](../components/breadcrumb.md)

**Common Use**: Showing page hierarchy, navigation trail, way-finding

### Swipebar

**Quick Description**: Swipeable navigation bar  
**Common Use**: Mobile navigation, gesture-based navigation

### Context Menu
**File**: [context-menu.md](../components/context-menu.md)  
**Quick Description**: Right-click/contextual menu  
**Common Use**: Context-sensitive actions, right-click menus
